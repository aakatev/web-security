# Microsoft Security Development Lifecycle (SDL)

> The SDL consists of a set of practices that support security assurance and compliance requirements. The SDL helps developers build more secure software by reducing the number and severity of vulnerabilities in software, while reducing development cost.

[Learn more](https://www.microsoft.com/en-us/securityengineering/sdl/practices) about MS SDL and some of the practices they recommend.
